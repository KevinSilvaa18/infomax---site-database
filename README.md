Infomax Enterprise OS

Bem-vindo ao repositório oficial do Infomax Enterprise OS, uma solução integrada de gestão para assistência técnica, focada em eficiência, rastreabilidade e gestão financeira.

O sistema é composto por uma arquitetura Full-Stack, separando o backend de processamento da interface de usuário.

🚀 Sobre o Projeto

O Infomax Enterprise é um ERP (Enterprise Resource Planning) simplificado e poderoso, desenvolvido para digitalizar o fluxo operacional de uma assistência técnica:

Gestão de Ordens de Serviço (OS) com geração de código de rastreio para clientes.

Controle Financeiro automatizado com base nas OSs.

Gestão de Estoque e Fornecedores unificada.

Interface responsiva e otimizada para alto desempenho.

📂 Estrutura do Projeto

O repositório está organizado para facilitar a manutenção e escalabilidade:

SITE-INFOMAX/
├── backend/          # API e Lógica de Negócio (Java/Spring Boot)
└── frontend/         # Interface e Dashboard (HTML/JS/Tailwind)


🛠 Tecnologias Utilizadas

Backend

Java

Spring Boot

Frontend

HTML5

Tailwind CSS (Design System)

JavaScript (Vanilla)

LocalStorage API (Gerenciamento de estado persistente)

⚙️ Como executar localmente

Clone o repositório:

git clone https://github.com/SEU_USUARIO/infomax-erp.git


Frontend:

Navegue até a pasta frontend/.

Abra o arquivo admin.html no seu navegador ou utilize a extensão "Live Server" no VS Code para uma experiência de desenvolvimento em tempo real.

Backend:

Abra a pasta backend/ na sua IDE (IntelliJ ou Eclipse).

Certifique-se de que o Maven/Gradle está configurado.

Execute a classe principal InfomaxApplication.
